# AB-730 Glossary

| Term | Meaning |
|---|---|
| **Accountability** | Responsible AI principle: humans remain responsible for AI decisions |
| **Bing** | Microsoft search engine that grounds web Copilot |
| **Citation** | Link Copilot adds to its answer pointing at the source file or page |
| **Connector** | Pre-built integration in Copilot Studio / Power Platform |
| **Content credential** | Provenance metadata embedded in Copilot-generated images |
| **Copilot Agent** | Persistent task-oriented Copilot |
| **Copilot Chat** | Conversational interface (web, Teams, Outlook, Edge) |
| **Copilot Lab** | Microsoft's prompt gallery |
| **Copilot Studio** | Low-code platform to build custom Copilots |
| **DALL-E** | OpenAI image-generation model used by Microsoft Copilot |
| **Fairness** | Responsible AI principle: AI does not unfairly disadvantage groups |
| **GCSE pattern** | Prompt template: Goal, Context, Source, Expectations |
| **Generative AI** | AI that creates new content from a prompt |
| **GPT** | "Generative Pre-trained Transformer" |
| **Grounding** | Adding factual context (web, tenant, custom) to a model's input |
| **Hallucination** | Confident but factually wrong AI output |
| **Inclusiveness** | Responsible AI principle: AI works for everyone |
| **LLM** | Large Language Model |
| **M365 Copilot** | Licensed Copilot inside Word, Excel, PPT, Outlook, Teams |
| **Microsoft Copilot** | Free web Copilot at copilot.microsoft.com |
| **Microsoft Copilot Pro** | Consumer subscription adding Copilot to consumer Office apps |
| **Microsoft Graph** | API surface to your tenant's data (mail, files, chats) |
| **Plugin / Extension** | Add-on that expands Copilot grounding |
| **Prompt** | The input text you give Copilot |
| **Privacy** | Responsible AI principle: data is protected |
| **Purview** | Microsoft's compliance + information protection platform |
| **Reliability and safety** | Responsible AI principle: AI works as intended |
| **Responsible AI** | Microsoft's six-principle framework |
| **Sensitivity label** | Microsoft Purview label flowing through Copilot output |
| **Token** | Unit of text used by LLMs |
| **Training cutoff** | Latest date the model has training data for |
| **Transparency** | Responsible AI principle: users know they are using AI |
