# AB-730 Learn Summaries

> Microsoft Learn summaries grouped by domain.

## Domain 1 - Generative AI Fundamentals

**Generative AI for business users.** GenAI creates new content (text, images, code) from a prompt by predicting next tokens with a model trained on massive datasets. Microsoft offers a portfolio: free Microsoft Copilot (web-grounded), Microsoft 365 Copilot (tenant-grounded via Microsoft Graph), Copilot Studio (custom), Azure AI (developer), GitHub Copilot (code). Hallucination - confident but wrong output - is a structural risk; mitigate by grounding, citations, and verification. Microsoft's six responsible AI principles (fairness, reliability and safety, privacy and security, inclusiveness, transparency, accountability) frame how AI should be designed and used.

## Domain 2 - Manage Prompts and Conversations

**Prompt engineering for business.** Effective prompts include Goal, Context, Source, and Expectations (the GCSE pattern Microsoft teaches). Copilot conversations remember context within a chat, so iterate rather than starting over. Use refinement verbs ("shorter," "more formal," "add bullets") to nudge the output. Anti-patterns: vague verbs, missing source for factual questions, multiple goals in one prompt. Sensitive prompts are filtered; M365 Copilot honors Microsoft Purview sensitivity labels so confidential content stays confidential.

## Domain 3 - Draft and Analyze Content

**Copilot inside Microsoft 365.** Word: draft, rewrite, summarize, translate. Excel: analyze tables, suggest formulas, create charts (data must be a structured table). PowerPoint: generate slides from a doc, redesign, add speaker notes. Outlook: draft replies, summarize threads, schedule. Teams: summarize meetings, list action items, catch up. M365 Copilot grounds in the Microsoft Graph and **cites** every fact; click the citation to verify. Sensitivity labels propagate to output. For non-work / public research, Microsoft Copilot (web) is the choice.
